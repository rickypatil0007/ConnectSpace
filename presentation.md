# ConnectSpace
**Connecting People, Spaces & Communities**

BLUEPRINT IDEATHON 2026
Problem Statement 3 — Urban Loneliness Epidemic & Unused Public Spaces

- **20M+** Mumbai Citizens
- **1000+** Underutilised Spaces
- **500+** Corporates Need Venues

---

## The Problem

Problem Statement 3 — The Urban 'Loneliness Epidemic' & Unused Public Spaces

**Loneliness Crisis**
- Senior citizens isolated at home
- Migrant workers lack social bonds
- WFH professionals feel disconnected
- No platform to discover local events

**VS**

**Wasted Spaces**
- 1000+ idle community halls & grounds
- Corporates need venues for CSR events
- NGOs can't find proper infrastructure
- Citizens unaware of events near them

---

## Our Solution

**ConnectSpace — One Platform. Three Stakeholders.**
*All stakeholders connect on one AI-powered ecosystem*

**Citizens**
- Interest-based event discovery
- AI personalized recommendations
- QR-code ticket & reminders
- Community networking

**Corporates + NGOs**
- Post & manage CSR/social events
- Set budget, audience & locality
- AI-matched venue suggestions
- Analytics & impact reports

**Space Owners**
- List idle venues with amenities
- Receive event booking requests
- Compare & accept proposals
- Earn from unused property

---

## How It Works

**End-to-End Platform Workflow**

1. **Organizer Creates Event**
2. **Space Owners Bid Proposals**
3. **Venue Confirmed**
4. **AI Recommends to Users**
5. **Event Hosted & Rated**

**AI Recommendation Engine**
- Personal Interests + Location
- Previous Participation History
- Event Popularity + Age Group
- Available Seats & Timing

*Example: Chess fan in Kandivali → sees "Chess Tournament in Kandivali" first*

---

## Feasibility & Tech Stack

**Tech Stack**
- **Frontend:** React.js / Flutter (mobile)
- **Backend:** Node.js + REST API
- **Database:** PostgreSQL + Firebase
- **AI Engine:** Python (scikit-learn / TF-IDF)
- **Maps & Location:** Google Maps API
- **Payments:** Razorpay
- **QR Check-in:** QR code + Webhooks

**Revenue Model**
- **Space Booking:** 5–10% commission per booking
- **Organizer Fee:** Platform service charge
- **Ticket Commission:** % of paid event tickets
- **Featured Events:** Promoted visibility fee
- **Analytics (Future):** Premium subscription

*MVP Scope: Web platform → Mobile App → Analytics Dashboard → CSR Reporting Module*
